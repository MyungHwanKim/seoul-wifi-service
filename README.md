# seoul-wifi-service
사용자 기반의 서울시 공공 와이파이 정보 제공 웹 사이트

### 사용기술스택
- JAVA (JDK 1.8), lombok
- JDBC, SQLite
- jsp, css, html
- 서울시 공공 WIFI Open API

### 개발 기간
- 2022.05.31 ~ 2022.06.11
